# 進捗状況

## 現在のフェーズ

ローカル Web UI と LM Studio 実行経路が動作する MVP 段階です。圧縮前後の文章、トークン推定、文字数、比率、保持条件、リスクを確認できます。

## 実装済み

- Rust の共通圧縮コア、CLI、ローカル Web UI
- LM Studio の OpenAI 互換 API への接続
- 設定ファイルで選ぶモデル、プロファイル、ランタイム
- 日本語入力を中心とする圧縮テンプレート
- JSON Schema を使う構造化出力
- 不適切な出力を検査し、一度だけ修復を試みる経路
- 日本語対応のトークン概算、文字数、圧縮比の表示
- アプリが llama.cpp サーバーを管理する内部 LLM 実行経路
- ランタイム失敗時に原文を返す fail-open 挙動
