## Current blocker

The process backend is wired, but `local-runtime-binaries/llama-cli` and the configured GGUF model
files are not present yet. Until they exist, the CLI falls back to returning the
original prompt with `should_send_original=true`.

## Recommended next action

Install or place a compatible `llama-cli` binary at `local-runtime-binaries/llama-cli.exe` on
Windows, add the configured GGUF model files under `local-model-files/`, then run the CLI
against the real runtime.
